# Walmart-Sales-
This project analyzes Walmart sales data to identify sales trends, customer purchasing behavior, product performance, and business opportunities. The analysis was performed using SQL for data extraction and business queries, while Power BI was used to build an interactive dashboard for decision-making.


## 🛠 Tools & Technologies

- SQL (MySQL)
- Power BI
- Excel
- DAX
- Power Query

---

## 📊 Dashboard KPIs

- Total Sales
- Total Revenue
- Gross Income
- Average Rating
- Total Quantity Sold
- Customer Count

---


## 📈 Business Questions Solved

1. Which branch generated the highest revenue?
2. Which product line has the highest sales?
3. What are the most preferred payment methods?
4. Which city contributes the most revenue?
5. Monthly sales trend analysis.
6. Customer type comparison.
7. Gender-wise purchasing behavior.
8. Rating distribution.
9. Best-selling products.
10. Peak shopping hours.


---



## 💡 Key Insights

- Food & Beverages generated the highest revenue.
- E-wallet was the most preferred payment method.
- Branch C recorded the highest sales.
- Female customers contributed more revenue than male customers.
- January recorded the highest monthly sales.
